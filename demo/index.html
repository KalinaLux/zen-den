🛡️ Privacy & Security

How your data stays safe — in plain English

🔒

Your data never leaves your laptop.

Zen Den runs 100% locally on your machine — like Spotify or Slack. Nothing is hosted in the cloud. Nothing is uploaded anywhere.

💻

Runs on Your Machine

The entire app is a local program. Your campaign data, API keys, and conversation history live in files on your Mac — not on anyone's server.

✓ No cloud hosting
✓ No external database
✓ Works offline (except live API queries)
🔗

Direct Google Connection

When the app checks a campaign, it talks directly to Google's servers using your own login — the same encrypted connection Google Ads already uses. No middleman.

✓ Your credentials, your connection
✓ Read-only access — can never modify campaigns
✓ Same security as google.com
🧠

AI Sees Questions, Not Data

The AI (ChatGPT/Claude) only receives your question — like "is the campaign on?" A separate local script fetches the answer from Google Ads. The AI formats the response but never sees raw account data, billing info, or anything sensitive.

✓ AI gets the question only
✓ Data fetched by local Python script
✓ No client PII sent to AI
🚫

Never Used to Train AI

Because you use the API (not the ChatGPT website), both OpenAI and Anthropic contractually guarantee your data is:

✓ Not used to train their AI models — ever
✓ Not stored beyond 30 days (abuse monitoring only)
✓ Not seen by any human at OpenAI or Anthropic

This is different from free ChatGPT. The API has strict business-grade data policies.

📋

Full Audit Trail

Every question asked and every action taken is logged locally. You can review the audit log anytime, and share it with your security team to prove data is handled responsibly.

✓ Every query timestamped
✓ Logs stored locally only
✓ Go to Audit Log to review
🌿

Local-Only Mode

Want even more privacy? The Zen Den can run in local-only mode where the AI is never called at all — it just uses the pre-loaded Q&A library and direct API queries. Zero data leaves your machine, period.

✓ Toggle in Settings anytime
✓ Q&A library works 100% offline
✓ Google Ads queries go direct
